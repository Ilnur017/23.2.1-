Данный проект представляет собой парсер для сбора оценок фильмов, выставленных пользователем на сайте Кинопоиск. Целью проекта является разработка инструмента, который позволяет:

1. Получать все оценки фильмов, выставленные конкретным пользователем, по его уникальному идентификатору (ID).
2. Фильтровать фильмы с рейтингом 8 и выше для дальнейшего анализа и создания рекомендаций.
3. Сохранять собранные данные в Excel файлы для последующей обработки.

Собранные данные могут быть использованы для анализа предпочтений пользователей и создания рекомендательных систем.

Инструкция по сборке и запуску

1. **Установите необходимые библиотеки:**

   Убедитесь, что у вас установлены библиотеки `requests`, `beautifulsoup4` и `pandas`. Вы можете установить их с помощью следующей команды:
   ```bash
   pip install requests beautifulsoup4 pandas openpyxl
